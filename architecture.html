Architecture & Pipeline
PHASE 1 WORLD GENERATION PYTHON
01
📍
USER INPUT
User provides a location — place name, GPS coordinates, or a Google Maps Street View URL.
02
🌐
GOOGLE MAPS API extract_tiles.py
Fetches the Street View 360° panorama for the location. Discovers nearby panoramas, picks the best one (highest resolution, least sky). Stitches a full equirectangular image, then slices it into 6 perspective tiles at 90° FOV each.
Places API Map Tiles API Street View Static API
PARALLEL — FROM PRESENT TILES
🏙️
PRESENT
Original tiles pass through directly — no editing needed. Full resolution from Street View.
🏛️
PAST · 1920s
Gemini 3 Pro image editing with strict camera-lock prompts. Transforms surfaces to cobblestone, Model Ts, gas lamps, period signage. Fed with reference images of historical architecture. generate_past.py
🚀
FUTURE · 2150
Gemini 3 Pro image editing. Same camera lock — transforms to flying vehicles, holographic displays, vertical gardens, smart surfaces. Clean, optimistic aesthetic. generate_future.py
04
🔬
RESOLUTION ENHANCEMENT
Gemini outputs at ~1070px — too low for sharp Gaussian splats. FLUX.2 Pro regenerates tiles at higher resolution with canny conditioning for structure lock. Real-ESRGAN upscales as a fallback path (2× super-resolution).
FLUX.2 Pro Real-ESRGAN Replicate / fal.ai
05
✨
WORLD LABS MARBLE API create_world.py
Each set of tiles is fed to World Labs to generate a navigable Gaussian splat world. Supports up to 4 images per world with azimuth hints. Outputs compressed .SPZ files (~20–30 MB each) — photorealistic 3D environments.
Marble 0.1-mini Gaussian Splatting .SPZ Format
PHASE 2 WEBXR EXPERIENCE TYPESCRIPT
06
🥽
IMMERSIVE WEBXR APP
Loads all three Gaussian splat worlds into an immersive WebXR scene. IWSDK provides the ECS framework with locomotion, grabbing, and spatial UI panels. SparkJS 2.0 handles GPU-accelerated splat rendering with LOD and foveation. Runs on Quest 3, Pico, or any WebXR browser.
IWSDK 0.2 SparkJS 2.0 Three.js Vite HTTPS
07
🌀
WORMHOLE TRANSITION
When switching eras: a fullscreen video sphere (cylinder + caps) envelops the viewer, playing a cosmic wormhole at 2.5× speed. A white flash fires as the new splat loads behind the effect. Ambient audio crossfades between era-specific tracks. Wormhole transition audio plays simultaneously. The old world unloads, the new one streams in.
Video Sphere ShaderMaterial Audio Crossfade
08
🤖
CONVAI AI AGENTS
A space maintenance robot (.glb model) floats in each era with a procedural idle animation (bob, tilt, drift). Powered by 3 individual Convai characters — each with their own ID, personality, and period-specific knowledge. Voice interaction via push-to-talk (grip) or toggle (X button) with gRPC streaming for real-time conversation.
Convai Web SDK gRPC Streaming 3 Character IDs .glb Model
